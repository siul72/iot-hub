Introduction

Webmanager is a java library for collection data from trains connected over TCP socket
The initial version is to be used with Online Diagnostics System Web Application
The main features are:

  • Collects process data
  • Collects event data
  • Collects environment data from events
  • Store event and environment data to a database
  • Store Data Snapshot to a database
  • Provides the enterprise entities to the web application
  • Provides authentication and security to web application access
  • Provides system configuration interface
  • Provides translations interface
